# AskIt
Статическая верстка Q&A сайта

## Запуск
```
python -m http.server 8000
Открыть http://localhost:8000/public
```

## Структура репозитория
```
askit/
├── public/
│   ├── base.html
│   ├── index.html
│   ├── question.html
│   ├── ask.html
│   ├── login.html
│   ├── signup.html
│   └── profile.html
├── public/static/
│   ├── css/
│   │   ├── bootstrap.min.css
│   │   └── styles.css
│   ├── js/
│   │   ├── bootstrap.bundle.min.js
│   │   └── main.js
│   └── img/
│       └── avatar.png
├── .gitignore
└── README.md
```
